2. Implementa como Aplicación web.
3. Sube `index.html` a GitHub Pages como archivo raíz.
4. Comparte la URL de GitHub Pages.

## Nota
Si vuelves a implementar Apps Script y cambia la URL `/exec`, tendrás que actualizar esa línea dentro de `index.html`.
